# 🗝️ OrbitKey

[![Platform](https://img.shields.io/badge/Platform-purpur-violet?logo=purpur)](https://purpurmc.org)
[![Java](https://shields.io)](https://oracle.com)
[![License](https://img.shields.io/badge/License-MIT-blue?logo=github)](LICENSE)

**OrbitKey** — это легковесный, полностью асинхронный плагин-утилита для игровых серверов Minecraft (Purpur/Paper 1.21.1+), предназначенный для безопасной передачи статуса сервера и текущего онлайна на ваш веб-сайт.

Плагин решает проблему долгого ожидания ответа веб-страниц (тайм-аутов) при стандартных query-запросах, полностью исключая задержки (фризы тиков) на стороне сервера.

---

## ✨ Особенности

* **⚡ Полная асинхронность:** Все HTTP-запросы и операции записи в базу данных выполняются в отдельных потоках. Никаких просадок TPS.
* **🔒 Криптографическая защита (HMAC-SHA256):** Запросы подписываются уникальным цифровым ключом. Сайт принимает данные только от вашего сервера, исключая подделку онлайна третьими лицами.
* **🛡️ Защита от Replay-атак:** Каждый пакет содержит Unix-таймштамп. Скрипт на сайте автоматически сбрасывает устаревшие или перехваченные повторные запросы.
* **📦 Встроенная H2 База Данных:** Плагин локально логирует состояние сети. Вы можете отслеживать стабильность связи прямо в игре.
* **🎨 Поддержка MiniMessage:** Современные градиенты, стили чата и интерактивные элементы для администраторов сервера.
* **🌐 Мультиязычность:** Готовые локализации (`ru_ru`, `en_us`, `uk_ua`).
* **🤖 Автогенерация:** При первом запуске плагин сам создаст защищенный 64-значный токен и готовый `.php` шаблон для вашего сайта.

## 🛠️ Настройка и установка

1. Скомпилируйте проект с помощью Maven (`mvn clean package`) и поместите `OrbitKey-1.0.0.jar` в папку `plugins/`.
2. Запустите сервер для генерации файлов конфигурации.
3. Зайдите в папку `plugins/OrbitKey/` и скопируйте автоматически сгенерированный файл `web_template.php`.
4. Загрузите этот файл на свой веб-хостинг (например, переименовав его в `status.php`).
5. В файле `plugins/OrbitKey/config.yml` укажите URL до вашего php-скрипта:
   ```yaml
   url: "https://your-site.com"
   ```
6. Выполните команду `/orbit key reload`.

---

## ⚙️ Конфигурация (`config.yml`)

```yaml
url: "https://server.com" # URL вашего скрипта на сайте
token: "" # Уникальный 64-значный ключ (сгенерируется сам, если оставить пустым)
language: "ru_ru" # Файл локализации из папки language/ (ru_ru, en_us, uk_ua)
debug-mode: 1 # Режим дебага в консоль и ОПам (0 - минимум, 1 - ошибки, 2 - полный лог)
logger-mode: 1 # Режим записи событий в локальную базу данных
logger-type: "H2" # Тип СУБД для логирования (на данный момент поддерживается H2 локально)
```

## 📋 Команды и Права

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/orbit key info` | Информация о плагине, авторе и версии. | `orbit.key.info` или `orbit.key.use` |
| `/orbit key site` | Показывает текущую ссылку на веб-ресурс сервера. | `orbit.key.site` или `orbit.key.use` |
| `/orbit key reload` | Перезагружает конфигурацию и языковые файлы. | `orbit.key.reload` |
| `/orbit key status` | Выводит состояние сети, аптайм и последние 3 лога из H2. | `orbit.key.status` |

*Игроки со статусом оператора (`/op`) имеют доступ ко всем командам по умолчанию.*
